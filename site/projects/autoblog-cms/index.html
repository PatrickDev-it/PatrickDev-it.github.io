Engineering thesis

A credible CMS is a workflow system.

The flagship journey spans authenticated access, workspace-derived policy, drafting, immutable revisions, optimistic concurrency, review, publication, and recovery. AI assistance and media processing are protected external failure domains rather than privileged shortcuts.

The same application logic must serve local review, automated tests, and configured deployment environments.
