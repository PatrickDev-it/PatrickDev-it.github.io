Machine learning · Scientific evidence · Project 03

Product Image Classifier

An image-classification system being converted from architectural claims into a reproducible experiment and inference platform.

Evaluation hardeningPublic repository ↗

Portfolio responsibility

This project demonstrates scientific engineering discipline: dataset provenance, split policy, experiment configuration, defensible baselines, calibration, error slicing, artifact traceability, and honest limitations.

Target evidence

  • Versioned dataset manifest, licensing, preprocessing, split policy, and seeds.
  • Baseline comparison and reproducible experiment configuration.
  • Accuracy, class-wise errors, calibration, confidence intervals, and failure examples.
  • Model and data cards tied to immutable artifacts.
  • CPU inference as the baseline plus a separately validated GPU profile.
  • Contract-tested service and batch inference interfaces.

Principal invariant

No performance number is published without the dataset, split, environment, method, and command required to reproduce it.

Current state

Existing production-readiness and expected-accuracy language is treated as unverified until the evaluation pipeline produces evidence. The task is image classification unless localization behavior is implemented and measured.
